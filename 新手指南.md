# RocketSim3D 操作指南

欢迎使用 RocketSim3D！本游戏分为「火箭组装（Workshop）」和「飞行模拟（Flight）」两大核心模块。以下是详细的操作指引。

## 🚀 飞行模拟模式 (Flight Mode)

### 火箭控制 (Rocket Controls)
*   **Space (空格键)**: 点火升空 (Ignition) / 级间分离 (如果可用)
*   **G**: 手动分离当前载荷段/火箭级 (Separate Stage)
*   **W / S**: 俯仰控制 (Pitch Up / Down)
*   **A / D**: 偏航控制 (Yaw Left / Right)
*   **Q / E**: 滚转控制 (Roll Left / Right)
*   **Z**: 油门全开 (Max Throttle)
*   **X**: 关闭油门 (Kill Throttle)
*   **Left Shift**: 增加油门 (Throttle Up)
*   **Left Ctrl**: 减少油门 (Throttle Down)

### 系统与自动驾驶 (Systems & Autopilot)
*   **TAB**: 切换 自动驾驶 / 手动控制 模式 (Auto / Manual Mode)
*   **T**: 开启/关闭 SAS 姿态保持辅助系统 (Stability Assist System)
*   **R**: 开启/关闭 RCS 姿态控制推力器 (Reaction Control System)
*   **N**: 开启/关闭 变轨节点自动执行 (Maneuver Auto-Execute)

### 视图与相机 (Camera & View)
*   **C**: 切换相机视角模式 (Cycle Camera Mode，分为自由视角、跟踪视角、系统全景等)
*   **O**: 开启/关闭 轨道显示 (Toggle Orbit Display)
*   **H**: 显示/隐藏 HUD 界面 (Toggle HUD)
*   **K**: 切换轨道参考系 (太阳中心 / 行星中心)
*   **鼠标滚轮**: 缩放视角
*   **Q / E (自由相机模式下)**: 倾斜/旋转镜头视角
*   **, (逗号) / . (句号)**: 切换相机对焦的目标天体

### 时间流逝 (Time Warp)
*   **1 / 2 / 3 / 4**: 时间加速控制 (分别为 1x, 10x, 100x, 1000x)
*(注: 在部分视角或轨道界面下，还支持最高至千万倍的时间加速)*

### 地形与环境 (Environment & SVO Terrain)
*   **Shift + P**: 开启/关闭 体积云 (Toggle Clouds)
*   **Shift + V**: 激活/解除 SVO 物理地形生成系统
*   **B**: 切换 SVO 地形编辑模式 (关闭 / 挖掘 Dig / 填筑 Build)
*   **鼠标左键 (在 Dig/Build 模式下)**: 执行挖掘或建造操作

---

## 🛠️ 火箭组装工厂 (Workshop Mode)

### 视角控制
*   **鼠标右键 (按住拖拽)**: 旋转环绕相机视角
*   **鼠标滚轮**: 垂直平移视角 (上下移动相机)
*   **Shift + 鼠标滚轮**: 缩放视角 (拉近 / 推远)

### 零件操作
*   **鼠标左键 (点击组件库)**: 选择并抓取新零件
*   **鼠标左键 (点击已有零件)**: 选中或挪动火箭部件
*   **Delete / Backspace**: 删除当前抓取在准星上的零件
*   **将零件拖向左侧组件库并点击**: 拖拽至左侧释放，可直接销毁该零件
*   *部分特殊控制(基于开发中的功能)*: 支持 W/A/S/D/Q/E 进行部件朝向的微调。重心(CoM)、气动中心(CoL)及推力中心(CoT) 可通过 HUD 的可视化切换。

---

## 🌠 变轨节点规划 (Maneuver Nodes)
在开启轨道显示 (O键) 及全景相机模式时：
*   **鼠标左键 (点击轨道)**: 创建变轨规划节点 (Maneuver Node)
*   **拖拽时间滑块**: 改变开始执行变轨的预计时间
*   **拖拽方向把手**: 调整 Prograde(顺行)/Retrograde(逆行)、Normal/Anti-Normal 方向的 Delta-V 增量。
*   *(规划完毕后并对齐变轨方向，按 N 可激活自动变轨执行功能)*

> **Tips**: 起飞前请确保推重比(TWR)大于 1。进入太空轨道的关键在于：升空后按一定角度(通常向东)逐渐倾斜(重力转向 Gravity Turn)，在达到目标高度的远地点(Apoapsis)时执行圆化变轨加速！
